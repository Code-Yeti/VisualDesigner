Topology Notes

Each site's Sophos XGS 450 firewall has its own dedicated 1Gb internet connection — internet traffic for a site never transits another site.

All three sites run an Aruba 8310 core switch. Inter-site traffic between Wise and Stroud routes through the Filton core switch, which acts as the central hub.

The DMZ connects directly to the Stroud firewall, isolated from the Wise/Filton inter-site trunk.
